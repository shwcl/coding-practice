
Project Brief
The application is used by an admin to manage IT requests and issues report by users. 

Tools used
It was built using PHP and MySQL for business logic and data store and HTML and CSS as the frontend languages.

Features and functionalities
1. It contains login/logout functionality
2. Allows a user to view open, pending and all tickets with pagination feature
3. Allows a user to create and edit tickets
4. Allows to view, add , edit ticket categories, users, and departments
5. Includes feature to allow search tickets across multiple fields in a single search
6. Includes pagination
